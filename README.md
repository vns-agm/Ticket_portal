# Ticket Portal

A personal issue tracking and ticket management app. Raise tickets, assign tasks, and monitor progress in one place.

**Developed by [Agam Srivastava](https://agam-portfolio.web.app/)** — personal project.

---

## Product description

**Short (for Vercel / meta / one-liner):**  
Ticket Portal is a personal project by Agam Srivastava: a simple issue tracker to create tickets, set priority and due dates, add comments, and close issues—all in one place.

**Full:**  
Ticket Portal is a lightweight issue and ticket management app built as a personal project by Agam Srivastava. Create and track issues with descriptions, priority levels, due dates, and assignees. Add comments to tickets and close them when done. Built with Next.js, React, and TypeScript for a fast, modern experience. No sign-up required—use it to manage tasks, bugs, or support requests for your own projects.

## Getting Started

Install dependencies and run the dev server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` — start development server
- `npm run build` — build for production
- `npm run start` — start production server
- `npm run lint` — run ESLint

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
